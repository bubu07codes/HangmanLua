# How to Play

1. **Download the Game**  
   Click on `Code` -> `Download ZIP`.

2. **Extract the Files**  
   Locate the downloaded ZIP file and extract its contents.

3. **Open the Folder**  
   Navigate to the extracted folder.

4. **Run the Game**  
   Double-click the `.exe` file to start playing.

---

**Thank You for Playing!**
